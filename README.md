# 🎲 Ludo Game

A simple interactive Ludo game built using React where players take turns rolling dice and tracking their moves.

## 🚀 Features
- Four players:
  - 🔴 Red
  - 🟡 Yellow
  - 🟢 Green
  - 🔵 Blue
- Separate Roll button for each player
- Move counter updates dynamically using React state
- Interactive UI with styled player cards
- Beginner-friendly React project for learning state updates

## 🛠️ Built With
- React
- JavaScript
- CSS / Bootstrap (if used)

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Gaurav-8530/ludo-game.git
```

Move into project folder:

```bash
cd ludo-game
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

## 🎮 How to Play
- Click any player's **Roll Dice** button.
- Each click updates that player's moves.
- Compete to see who reaches the highest moves!
